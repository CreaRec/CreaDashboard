# CreaDashboard

Home dashboard for utility expenses, calendar events, reminders, and notes.

## Stack

- **Frontend:** React + Webpack + TypeScript + Tailwind CSS + Recharts
- **Backend:** Node.js + Express + TypeScript

## Getting started

```bash
npm install
npm run dev
```

- Frontend: http://localhost:3000
- Backend API: http://localhost:3001

## API endpoints (stubs)

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/utilities` | Utility consumption data |
| GET | `/api/calendar` | Calendar events |
| GET | `/api/reminders` | Reminders list |
| GET | `/api/notes` | Notes list |

## Project structure

CreaDashboard/
